# Level 2 - Create Our First React App

## Overview

This section explains how to create and run a React project using modern development tools.

The project is built using Vite because it provides a faster development experience than Create React App.

---

## Topics Covered

- Create React App
- Vite
- npm
- npx
- Node.js
- Project Structure
- JSX
- React Components
- JSX Rules
- JavaScript inside JSX
- className
- htmlFor

---

## Folder Structure

```
src
│
├── main.jsx
├── App.jsx
└── assets
```

---

## JSX Rules

- Return only one parent element
- Use className instead of class
- Use htmlFor instead of for
- JavaScript is written inside {}

---

## Example

```jsx
function App() {
  const name = "Alok";

  return <h1>Hello {name}</h1>;
}
```
