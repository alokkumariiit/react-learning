# Level 5 - React Router

## Overview

React Router enables navigation between pages without refreshing the browser.

It is used to build Single Page Applications (SPA).

---

## Topics Covered

- BrowserRouter
- Routes
- Route
- Link
- NavLink
- Nested Routes
- Dynamic Routes
- URL Parameters
- 404 Pages

---

## Why React Router?

Instead of loading a new HTML page every time,

React changes only the component.

This makes applications much faster.

---

## Example

```
/
|
|-- Home
|-- About
|-- Contact
|-- Services
```

---

## Outcome

You'll be able to build multi-page React applications using routing.
